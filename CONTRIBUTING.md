# Contributing

## Have questions about any part of the project?

Participating in this project is not limited to writing code, your usage and feedback are also invaluable contributions!

If you find any bugs or have any feature requests, open an issue to let us know, we'll try our best to help you.

## Want to implement something yourself?

Have a look at existing issues and pick what you're interested in, especially those with the "good first issue" label, such issues are relatively easy to implement.

For more detailed guides, see the [contributing](https://ex-actor.github.io/ex-actor/contributing/) page for details like [how to build from source](https://ex-actor.github.io/ex-actor/contributing/#how-to-build-from-source).
